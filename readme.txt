Página web para revisar audios de radios junto a radiologger

RadioLogger va a ser montado en un server intermedio (bridge) que:
1. Ejecute RadioLogger en el servidor (una instancia por radio/IP)
2. Guarde los audios que genera (MP3/WAV)
3. Los suba automáticamente a la plataforma web, que ya los clasifica por radio, ciudad, fecha y horario
